В репозитории находится три папки:
1) Папка first_solution содержит пробное решение (без использования pytest и PageObject)
2) В папке Tensor_task1 находится решение первого сценария, для запуска скачать данную папку, поместить в папку PycharmObjects, открыть в PyCharm и в терминале ввести pytest tests/test_saby.py -s
3) В папке Tensor_task2 находится решение второго сценария, для запуска скачать данную папку, поместить в папку PycharmObjects, открыть в PyCharm и в терминале ввести pytest tests/test_contacts.py -s
